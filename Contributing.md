# Contributing to Xypheria Membrane EmployeMe

Thanks for your interest in contributing! 🎉

## How to Contribute

1. Fork the repository
2. Create your feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Commit your changes:
   ```bash
   git commit -m "feat: add your feature"
   ```
4. Push to your branch:
   ```bash
   git push origin feature/your-feature-name
   ```
5. Open a Pull Request

---

## Development Setup

### Web (Vite)
```bash
npm install
npm run dev
```

### Flutter
```bash
cd employme_flutter
flutter pub get
flutter run
```

---

## Commit Convention (Recommended)

Use conventional commit style:

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `refactor:` code refactor
- `test:` tests
- `chore:` maintenance

Example:
```bash
git commit -m "fix: resolve employee list pagination bug"
```

---

## Pull Request Guidelines

- Keep PRs focused and small
- Add/update tests when possible
- Update documentation if behavior changes
- Ensure app builds successfully before PR

---

## Code Style

- Follow existing code style in the project
- Keep functions and widgets/components readable
- Prefer meaningful variable/function names

Thanks for helping improve this project 💙
